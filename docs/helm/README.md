# ORY Helm Charts

Add this repository using:

```bash
$ helm repo add ory https://k8s.ory.sh/helm/charts
$ helm repo update
```

Documentation on each individual chart can be found for

- [ORY Kratos Helm Chart](./kratos.md)
- [ORY Hydra Helm Chart](./hydra.md)
